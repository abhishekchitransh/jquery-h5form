jQuery.h5form - HTML5 Forms Plugin

This plugin gives all browsers the HTML5 forms like the Opera.

These enhancements are determined in the presence or absence of the object in a input element. Therefore it will be able to support nearly maintenance-free for the other versions of browsers.

Some of the features like the date state use to determine the type and version of the browser. Even if the browser improvements, it uses the extension of the plugin (2.9.0).
Because, it is in order to avoid the maintenance support by fixing the state it was in when you install the plugin to the site.

And this will also fix several bugs of the Internet Explorer on submit, and will allow you to use the button element.

This has the large file size for multi-function. However, it binds only the features that are needed by the browser for the element of the page, and it will work lightly.
If you wish, you can get a smaller size javascript by removing unnecessary features.

Installation

You may only append to your HTML header the following code.
Please rewrite the URL path as appropriate. 

<link rel="stylesheet" type="text/css" href="css/jquery.h5form--laster.css" />
<script type="text/javascript"
  src="https://ajax.googleapis.com/ajax/libs/jquery/1.8.3/jquery.min.js"></script>
<script type="text/javascript"
  src="jquery.h5form--laster.min.js"></script>

<script type="text/javascript">
$(function() {
  $('form').h5form();
});
</script>

CSS

If you have styles that specified by type of the number, date or time , please add class name "h5form-typename".
The elements that have been extended by the plugin is corrected type and added the class instead. 

[type="number"], .h5form-number {
	font-weight: bold;
}

Options

You can specify the options in the table below to the h5form, if necessary. 

$('form').h5form({
	datepicker: { showButtonPanel: true },
	dynamicHtml: '[name="password2"]'
});

Option name Defalut value Description
styleErr { backgroundColor: 'mistyrose' } The style of the validation errors. It does nothing if you set false.
colorOff '#a1a1a1' The color of the placeholder.
msgEmpty 'Please enter this field.' The error messages.

In the msgMaxlen, msgMin and msgMax, # is replaced by the value.
If the element has a title attribute, it is appended to these messages.
msgUnselct 'Please select an item.'
msgUncheck 'Please check this checkbox.'
msgPattern 'Does not match the required pattern.'
msgEmail 'E-mail address is not correct.'
msgUrl 'URL is not correct.'
msgMaxlen 'Too many # characters.'
msgInvalid 'Value is invalid.'
msgMin 'Please be greater than or equal to #.'
msgMax 'Please be less than or equal to #.'
datepicker { } The options of datepicker. See the documents of the jQuery UI.
However, the maxDate and minDate are determined by the min and max attributes of the element, and the value of the dateFormat is fixed at 'yy-mm-dd'.
dynamicHtml '.h5form-dynamic' For example, if you have a script to add the required attribute to email2 when the email has been entered, you may set 'h5form-dynamic' to the class of email2.
Then, validation is performed at submission. Or, you may specify the selector of the target element to this option.
exprBehide '.h5form-behind' If you have the hidden balloons behind the other element, you may set 'h5form-behind' to the class of the element, or specify the selector to this option.
The element is in a disabled state while displayed the balloon.

You will find some of the other options on the source. Please note the undocumented options will be changed or discontinued without notice.

Balloon messages

The balloon message is simply inserted just before each element.
Then relative to the display position of the each element, it place shift the container and inner, and expresses the beard.

Also, the error message on the server side after submit transition, you can get the balloon message by the same output format.
