jQuery.h5form - HTML5 Forms Plugin

This plugin gives all browsers the HTML5 forms like the Opera.

Whether these enhancements are determined in the presence or absence of the object in a input element.
For the other versions of browsers, it will be able to support nearly maintenance-free ... maybe.

And this will also fix several bugs of the Internet Explorer on submit, and will allow you to use the button element.

Installation

You may only append to your HTML header the following code.
Please rewrite the URL path as appropriate. 

<link rel="stylesheet" type="text/css" href="css/jquery.h5form--laster.css" />
<script type="text/javascript"
  src="https://ajax.googleapis.com/ajax/libs/jquery/1.8.3/jquery.min.js"></script>
<script type="text/javascript"
  src="jquery.h5form--laster.min.js"></script>

<script type="text/javascript">
$(function() {
  $('form').h5form();
});
</script>

jQuery UI

If you wisth, please install the jQeury UI for the date and range state and autocomplete.

<link rel="stylesheet" type="text/css"
  href="../../css/smoothness/jquery-ui-custom.css" />
<script type="text/javascript"
  src="https://ajax.googleapis.com/ajax/libs/jqueryui/1.9.2/jquery-ui.min.js"></script>

On the element of the date or range state, the datepicker or the slider will be automatically handle.
Because the judgment the existence of the jQury UI, you do not need to enter additional settings.

Options

If you have a dynamic html element that requires a validation, please add the selector to the option "dynamicHtml".

$('form').h5form({ dynamicHtml: '.h5form-dynamic' });	// default

Disabled spin button of Number and time in default. Please add following option if you wish.
This is because the design is not to collapse even duplicate the functionality of the browser by halfway version. 

$('form').h5form({ addSpin: true });

There is a bug in the Z-index of Android.
If you have a hidden balloon behind other element, you may set this option. This optional element is in a disabled state while displayed the balloon. 

$('form').h5form({ exprBehide: '.h5form-behind' });	// default
