jQuery.h5form - HTML5 Forms Plugin

すべてのブラウザに Opera のような HTML5 フォーム機能を提供するプラグインです。

これらの拡張は input 要素がオブジェクトを持っているかどうかで判定しています。
ブラウザがバージョンアップしても、ほぼメンテナンスフリーで対応できるでしょう（たぶん）

また Internet Explorer のバグ仕様も改善しましたので、button 要素が気兼ねなく利用できるようになります。

*1
IE 6 ではフキダシが select 要素の背後に隠れます。
*2
グローバルの日付の仕様は W3C のドラフトの準拠しました。 このため、Opera とは異なる ユーザインタフェース となってしまいました。
*3
jQuery UI を使えば対応は難しくありません。これらの状態も必要ですか？
*4
jQuery UI にはカラーウィジットがなく、標準と呼べるプラグインがありません。
*5
HTML5 ではフォーム関連の要素はすべて form 属性を持っていますが、このプラグインはサブミットボタンのみサポートします。
仕様書通りの働きにはならず、また他の機能のバグの温床になるため、この機能は削除しました。
*6
この問題を javascript で解決するのは不可能なようです。
*7
IE 10 では HTML5 フォームがサポートそうです。 しかし、どうやらそれはバリデーションだけの実装であるようです。
もしそうなら、プラグインは修正なしで number その他の属性のインターフェースを提供することはできません。
*8
Android ブラウザは HTML5 の属性をオブジェクトに持っていながら、その機能は実装されていません。 HTML5 フォーム解説サイトの多くは間違った情報を流してるので注意してください。
私は自分の Android 2.3.4 スマートフォンでこれを確認しました。他のバージョンや iPhone についての情報があればお知らせください。
*9
プラグインは min や max step 属性のバリデーションのみを提供し、スピンボタンなどインターフェースは追加しません。
*10
IE 9 は Datalist 要素の Option 子要素を DOM に読み込みません。IE 9でも動作するようにするには、メタタグ X-UA-Compatible で IE=8 を指定しなければなりません。

アーカイブとソースは GitHub からダウンロードできます。

https://github.com/rapidexp/jquery-h5form

不要な機能を省いたより小さな javascript もダウンロードできます。 不要な機能のチェックを外してから ダウンロード ボタンをクリックしてください。
この他にアーカイブに含まれる スタイルシート は別途のダウンロードが必要です。

Installation

次のコードを HTML ヘッダに加えるだけです。
URL パスは適時読み替えてください。 

<link rel="stylesheet" type="text/css" href="css/jquery.h5form--laster.css" />
<script type="text/javascript"
  src="https://ajax.googleapis.com/ajax/libs/jquery/1.8.3/jquery.min.js"></script>
<script type="text/javascript"
  src="jquery.h5form--laster.min.js"></script>

<script type="text/javascript">
$(function() {
  $('form').h5form();
});
</script>

jQuery UI

Date や Number 状態のユーザインタフェース及び Autocomplete が必要なら jQeury UI をインストールしてください。

<link rel="stylesheet" type="text/css"
  href="../../css/smoothness/jquery-ui-custom.css" />
<script type="text/javascript"
  src="https://ajax.googleapis.com/ajax/libs/jqueryui/1.9.2/jquery-ui.min.js"></script>

Date や Range 状態の要素には、カレンダーやスライダーが自動的にハンドルされます。
jQury UI がインストールされているかどうかで判断されるので、他に必要な追加設定はありません。

Options

Required 属性が動的に変化がするエレメントがある場合には、dynamicHtml オプションでそれらの要素のセレクターを指定してください。

$('form').h5form({ dynamicHtml: '.h5form-dynamic' });	// default

NumberとTime属性のスピンボタンでデフォルトで無効にしました。これが必要なら次のオプションを指定してください。
ブラウザの中途半端な対応により機能が重複してしまったときに、デザインが崩れないための措置です。スピンボタンがない場合でも上下方向キーで数値を変更できます。

$('form').h5form({ addSpin: true });

Android には Z-index にバグがあります。 フキダシが他の要素の背後に隠れてしまう場合は、このオプションを設定してください。
ここで指定したセレクターの要素は、フキダシが表示されている間、disabled 状態になるように制御されます。 

$('form').h5form({ exprBehide: '.h5form-behind' });	// default
